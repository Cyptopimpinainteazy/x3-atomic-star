## Key Features

### Single Tool Calls
```python
result = caller.call_single_tool("Analyze file", tools=[analyze])
```
- Simple focused analysis
- Fast execution
- Real-time feedback option

### Parallel Tools
```python
result = caller.call_parallel_tools("Audit module", tools=[security, perf, tests])
```
- Multiple analyses simultaneously
- Model synthesis of results
- Streaming support

### Agent Loops
```python
result = caller.agent_loop("Calculate result", tools=[add, multiply], max_iterations=5)
```
- Multi-turn reasoning
- Automatic tool decisions
- Complex problem solving

### Streaming Mode
```
Real-time: Model Thinking... → Tool Calls... → Results Synthesis
```
- User sees output immediately
- Model reasoning visible
- Better perceived performance

### Non-Streaming Mode
```
Blocking until complete, simpler code paths
```
- Easier for CI/CD
- No streaming overhead
- Better for batch processing

## Quick Reference

### Import
```python
from x3_chain_ollama_tools import StreamingToolCaller, X3ChainAnalyzer
```

### Create Caller
```python
caller = StreamingToolCaller(model="qwen2.5-coder:7b")
```

### Three Patterns
```python
# Single
result = caller.call_single_tool(message, tools=[func])

# Parallel  
result = caller.call_parallel_tools(message, tools=[f1, f2, f3])

# Agent
result = caller.agent_loop(message, tools=[f1, f2], max_iterations=5)
```

### Access Results
```python
result.thinking        # Model's reasoning
result.content         # Initial response
result.tool_calls_made # [(name, args), ...]
result.final_response  # Final synthesized answer
```
